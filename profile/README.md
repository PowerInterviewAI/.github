# Power Interview AI

**Privacy-first AI interview assistant** for technical and behavioral interviews — built for real, live scenarios.

> Sensitive data stays **on your device** — secure local storage with user-controlled information. No conversation data is sent to third parties.

## What we're building

| Feature | Description |
|---|---|
| 🎙 **Live transcription (dual-channel)** | Speaker-aware transcription with full conversation history |
| 💡 **AI reply suggestions** | Personalized responses based on your CV, job description, and ongoing conversation |
| 💻 **Code suggestions** | Screenshot-based coding help with syntax-highlighted solutions |
| 🕶 **Stealth mode** | Discreet on-screen operation — hotkeys, opacity, positioning — non-capturable during screen share |
| 📄 **Smart export** | DOCX transcript export with AI summaries, insights, and communication-pattern analysis |
